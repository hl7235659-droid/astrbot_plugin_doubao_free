# astrbot_plugin_doubao_free

豆包（dola.com 国际版）免费 API 插件 for AstrBot。

填入 Cookie 即可使用：**聊天、图片识别、文生图、视频生成**，支持多账号轮询。

## 功能

| 指令 | 功能 | 示例 |
|---|---|---|
| `/豆包 <内容>` | 和豆包聊天 | `/豆包 你好` |
| `/识图 <问题>` | 识别图片（需附带图片） | 发图片 + `/识图 这是什么动物` |
| `/生图 <描述>` | 文生图 | `/生图 一只可爱的橘猫` |
| `/生成视频 <描述>` | 视频生成 | `/生成视频 一只小猫追蝴蝶 16:9 5秒` |
| `/视频 <描述>` | 视频生成（别名） | `/视频 海浪拍打礁石 9:16` |

## 安装

1. 在 AstrBot 插件市场搜索 `doubao_free` 安装，或手动将本插件放入 `data/plugins/` 目录
2. 重启 AstrBot
3. 在 AstrBot 管理面板的插件配置中，填入你的 dola.com Cookie

## 获取 Cookie

1. 打开 [www.dola.com](https://www.dola.com) 并登录（需要 JP/KR 等地区出口 IP）
2. 按 F12 打开开发者工具 → Network 标签
3. 刷新页面，点击任意请求 → Headers → 复制完整的 `Cookie` 值
4. 粘贴到插件配置的 `cookies` 列表中（每行一个 Cookie，支持多个账号轮询）

> Cookie 需包含 `msToken=`、`s_v_web_id=`、`sessionid=` 字段才有效。

## 视频参数

比例和时长可以直接写在描述里，也可用显式参数：

```
/生成视频 一只小猫追蝴蝶 16:9 10秒
/生成视频 一只小猫追蝴蝶 --ratio 16:9 --duration 5
```

- **比例**：`16:9` / `9:16` / `1:1` / `4:3` 等，默认 `9:16`
- **时长**：3-10 秒，默认 5 秒

视频生成约 2-4 分钟，生成后会自动推送到当前会话。

## 多账号轮询

- 配置多个 Cookie 会自动随机轮询
- 某个账号额度不足会自动换下一个（检测多语言失败文本，不等超时）
- 视频生成需要豆包国际版每日免费额度

## 限制

- **需要 JP/KR 等地区出口 IP**：dola.com 有地理限制，国内服务器需配代理
- **Cookie 会过期**：约 60 天，过期后需重新登录获取
- **不支持国内版**：本插件只支持 dola.com 国际版（doubao.com 国内版的 a_bogus 签名无法绕过）
